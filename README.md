# 🔐 SecureScan AI

SecureScan AI is a lightweight phishing & scam detection web app with an AI-powered assistant using Google Gemini.

---

## 🚀 Features

- 🔍 Scam / phishing keyword scanner
- 🧠 AI security assistant (Gemini-powered)
- ⚠ Risk scoring system (LOW / MEDIUM / HIGH)
- 💬 Chat-based security analysis
- 🌐 Modern responsive UI

---

## 🏗 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- AI: Google Gemini API
- Hosting: Render / any Node hosting
